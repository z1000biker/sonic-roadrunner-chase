# Sonic vs Road Runner Chase

An interactive 3D scene featuring Sonic the Hedgehog chasing Road Runner through dynamically transitioning environments.

## Features

- **Detailed Character Models**: Custom-built 3D models of Sonic and Road Runner with realistic animations
- **Dynamic Environments**: Seamless transition between forest and desert biomes
- **Infinite Terrain System**: Procedurally generated terrain that extends infinitely
- **Realistic Animations**: Smooth running animations with proper bone articulation
- **Visual Effects**: Dust particles, motion blur, and environmental effects

## Files

- `sonic-chase.html` - Main HTML entry point
- `sonic-chase.css` - Styling and layout
- `chase-main.js` - Main scene setup and animation loop
- `chase-sonic.js` - Sonic character model and animations
- `chase-roadrunner.js` - Road Runner character model and animations
- `chase-forest.js` - Forest environment generation
- `chase-desert.js` - Desert environment generation
- `chase-terrain.js` - Infinite terrain system
- `chase-effects.js` - Visual effects and particles

## How to Run

1. Clone this repository
2. Open `sonic-chase.html` in a modern web browser
3. The scene will load automatically with Sonic chasing Road Runner

## Technologies

- Three.js - 3D graphics library
- JavaScript ES6 modules
- HTML5 Canvas

## Controls

The scene runs automatically. Watch as Sonic chases Road Runner through the ever-changing landscape!
